Atalhos de VS Code

Atalhos gerais

Vamos ver alguns atalhos legais do VS code e vão deixar nossa vida mais alegre, supimpa e feliz.

altz+Z:Quebra automática de linha

alt+setas cima/baixo: move a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor está.

shift+alt+setas cima/baixo: copia a seleção para cima/baixo no arquivo. se nada estiver selecionado, copia a linha em que o cursor está.

ctrl+espaço: chama o autocompletar de um contexto.

ctrl+; : comentário/descomentario
